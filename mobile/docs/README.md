# Mobile

## Назначение

Мобильные приложения Android и iOS дают быстрый доступ к состоянию пасеки и уведомлениям.

## Стек

- React Native;
- TypeScript.

## Языки

Мобильные приложения должны поддерживать:

- русский;
- украинский;
- английский.

## Основные функции

- Авторизация пользователя.
- Просмотр пасек.
- Просмотр ульев.
- Карточка улья.
- Графики истории.
- Push-уведомления.
- Журнал событий.
- Настройки уведомлений.
- Настройки устройства.
- Привязка устройства к улью после MVP.

## MVP

Для MVP мобильное приложение должно:

- показывать текущее состояние улья или пасеки;
- получать MVP-набор push-уведомлений, если push-инфраструктура готова;
- работать с выбранным языком пользователя.

## Уведомления MVP

- Открытие улья.
- Резкое изменение веса.
- Потеря связи с устройством.
- Восстановление связи с устройством.
- Ошибка датчика.
- Низкий заряд батареи.

## После MVP

- Привязка устройства через QR-код.
- Поиск устройства в режиме настройки.
- Передача `deviceToken` на устройство после привязки.
- Расширенные настройки устройства.
- Удаленное изменение пинов, если будет разрешено в backend.
