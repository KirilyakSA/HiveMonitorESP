# HiveMonitor

HiveMonitor - комплексная система мониторинга ульев и пасеки.

Система состоит из IoT-устройств на ESP8266/ESP32, backend, базы данных, MQTT broker, web-интерфейса, мобильных приложений Android/iOS и инфраструктуры развертывания.

## Цели

- Управлять организациями, пасеками, ульями и устройствами.
- Собирать телеметрию с одного или нескольких ульев.
- Хранить raw-историю измерений и в будущем переносить старые данные в архив.
- Показывать текущее состояние ульев и историю показателей.
- Строить графики веса, температуры, влажности и погодных условий.
- Фиксировать открытие улья, события, задачи и результаты работ.
- Выявлять резкие изменения веса, ошибки датчиков и пропущенные плановые передачи.
- Отправлять push/Telegram/in-app уведомления в следующих backend-инкрементах.
- Поддерживать локальную настройку IoT-устройства через web-интерфейс.
- Поддерживать будущие premium-функции: weather analytics, AI, OTA, billing.
- Поддерживать web и mobile интерфейсы на русском, украинском и английском языках.
